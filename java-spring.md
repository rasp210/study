## Spring容器核心组件
```
1. beans：表示的是对spring以及所有bean对象的管理
2. core：包含了最底层的开发支持，例如:依赖的注入关系，资源文件的访问，数据类型的转换
3. context：提供的是一个完整的容器上下文，在这个上下文可以处理对象生命周期或者是事务
```
