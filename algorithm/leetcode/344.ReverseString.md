# 344.Reverse String

> 反转字符串   
> Write a function that takes a string as input and returns the string reversed.

**Example**
>Input: "hello"   
>Output: "olleh"  
>    
>Input: "A man, a plan, a canal: Panama"   
>Output: "amanaP :lanac a ,nalp a ,nam A"   

**Solution**
```c
char* reverseString(char* s) {
    int len = strlen(s);
    if (len <= 1) {
        return s;
    }
    
    char temp;
    for (int l = 0, r = len -1; l < r; l++, r--) {
        temp = s[l];
        s[l] = s[r];
        s[r] = temp;
    }
    return s;
}
```
